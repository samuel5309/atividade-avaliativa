## ⚙️ Como Clonar e Navegar no Repositório
Caso queira testar qualquer um dos projetos localmente, siga os passos abaixo no terminal:

1.  **Clonar o repositório:**
    ```bash
    git clone https://github.com/[seu-usuario]/[seu-repositorio-portfolio].git
    ```
2.  **Entrar no diretório:**
    ```bash
    cd [seu-repositorio-portfolio]
    ```
3.  Navegar até a pasta da disciplina desejada utilizando o comando `cd [nome-da-pasta]` e ler o `README.md` específico da pasta para instruções detalhadas de inicialização do projeto.
